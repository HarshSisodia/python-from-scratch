## 🚀 How to Run

Make sure Python is installed:

```bash
python --version
```

Clone the repository:

```bash
git clone YOUR_REPOSITORY_URL
```

Open the project folder:

```bash
cd python-from-scratch
```

Run any Python file:

```bash
python filename.py
```
